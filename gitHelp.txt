Команды для регистрации ваших данных 
git config --global user.name "Your name"
git config --global user.email "your-email@example.com"

Команды только для одного проекта 

1) Это Команды для инициализации Git в вашем новом проекте , 
пишется один раз в начале нового проекта:

git init

2) Эта команда будет вставить вашу ссылку на ваш репозиторий:

git remote add origin "ссылка на репозиотрий"


Основные комнады для работы с ГИТОМ

1) Проверка состояние нашего проекта 

git status

2) Скопировать в буфер 

git add .

3) Сохранить все что у нас в буфере 

git commit - m "first lesson save"

4) Отправка на сохранений в Github

git push origin main/master

5) Удаление всех изменений до коммита
git restore .

6) Удаление после git add.

git reset .
